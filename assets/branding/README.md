# Trix Branding Draft

- `trix-mark.svg`: standalone app mark.
- `trix-logo.svg`: primary lockup for light surfaces.
- `trix-logo-inverse.svg`: lockup for dark surfaces.
- `trix-preview.svg`: review sheet with both variants.

Palette:

- `#09131C`: dark anchor
- `#18D2FF`: electric cyan
- `#B6FF63`: acid lime accent

To regenerate platform launcher icons from `trix-mark.svg`:

```bash
swift scripts/generate-app-icons.swift
```
